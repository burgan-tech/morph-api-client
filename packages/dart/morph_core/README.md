# morph_core (Dart)

Scaffold for the Dart port of **`@morph/core`** (Morph API Client). See repository
[`docs/dart-parity.md`](../../../docs/dart-parity.md).

**Status:** `MorphClient.init` throws `UnimplementedError` until OAuth/HTTP parity
work lands (GitHub issue [#1](https://github.com/burgan-tech/morph-api-client/issues/1)).

```bash
dart pub get
dart analyze
dart test
```
